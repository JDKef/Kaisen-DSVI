# Kaisen

Sistema de Gestión de Inventario y Ventas desarrollado en Flutter.

## Integrantes

- Kevin Rodríguez

## Tecnologías

- Flutter
- Dart
- Supabase
- PostgreSQL
- Provider

## Funcionalidades

- Inicio de sesión
- Gestión de productos
- Gestión de inventario
- Registro de ventas
- Historial de ventas
- Escaneo de códigos de barras
- Dashboard

## Requisitos

- Flutter 3.44.6
- Dart 3.12.2

## Instalación

```bash
flutter pub get
```

Ejecutar:

```bash
flutter run
```

Configurando previamente las variables:

```
SUPABASE_URL
SUPABASE_PUBLISHABLE_KEY
```
