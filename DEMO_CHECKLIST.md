# Lista de verificación para la demostración de Kaisen DS6

- [ ] Instalar el APK proporcionado o abrir la aplicación desde Flutter.
- [ ] Registrar un nuevo usuario de prueba identificable o iniciar sesión con la cuenta de dicho usuario.
- [ ] Crear un producto de prueba con un nombre claro, stock y código de barras.
- [ ] Editar el producto y verificar los valores guardados.
- [ ] Abrir **Registrar venta** y escanear el código de barras del producto.
- [ ] Registrar una venta con una cantidad superior a uno.
- [ ] Volver al inventario y verificar la reducción de stock esperada.
- [ ] Abrir el historial de ventas y verificar la línea de venta y la cantidad.
- [ ] Actualizar las pantallas correspondientes y confirmar que los valores remotos siguen siendo visibles.
- [ ] Cerrar y volver a abrir la aplicación.
- [ ] Verificar que se restaure la sesión autenticada.
- [ ] Cerrar sesión y confirmar que reaparece la pantalla de inicio de sesión.
